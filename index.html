Profil Hansen Ferdinando Yosua Pontoh

Website ini saya buat dengan tujuan untuk mempermudah saya dalam mengakses dan membuka berbagai file yang saya miliki, serta mengarsipkannya dalam bentuk portofolio. Dengan adanya website ini, saya dapat dengan mudah menyimpan dan menampilkan berbagai informasi mengenai diri saya, keterampilan, dan karya yang telah saya buat.

Selain itu, website ini juga berfungsi sebagai media untuk membagikan data diri saya kepada orang lain. Jika ada yang membutuhkan informasi mengenai saya, seperti portofolio atau riwayat hidup, saya dapat dengan cepat memberikan akses ke website ini sebagai referensi. Dengan demikian, website ini membantu saya dalam menyajikan informasi secara profesional dan terorganisir.
